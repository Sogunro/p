## Component Library Recommendation

For rapid development, use a component library:

**Recommended:** shadcn/ui
- Built on Radix UI primitives
- Tailwind CSS styling
- Copy-paste components (not a dependency)
- Good for: buttons, modals, inputs, cards, checkboxes

**For Canvas/Whiteboard:**
- React Flow (for node connections and canvas)
- Or custom implementation with drag-and-drop
